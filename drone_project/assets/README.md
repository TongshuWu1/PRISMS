# Assets

Source mesh assets used to build the CoppeliaSim plant model.

- `meshes/crazyflie_cage_body_no_propellers.stl`: Crazyflie body and cage visual mesh.
- `meshes/crazyflie_propellers_aligned.stl`: propeller visual mesh aligned to the motor layout.

Keep raw/generated mesh inputs here. The runnable CoppeliaSim model belongs in
`model/`.
